## Main Repositories

 - [Pydio Cells](https://github.com/pydio/cells): Golang-based server with MUI3 ReactJS embedded frontend
 - [Cells Client](https://github.com/pydio/cells-client): CLI Tool for communicating with Pydio Cells REST API
